Déploiement d'un Honeypot

Défense Active | Septembre 2024

Description

Mise en place d'un "Pot de Miel" (Honeypot) simulant un serveur SSH vulnérable exposé sur Internet. Le but était de collecter des informations sur les techniques d'attaque par force brute (Brute Force).

Architecture

  • Déploiement sur un VPS cloud.
  • Utilisation de Docker pour isoler le service.
  • Centralisation des logs vers une stack ELK (Elasticsearch, Logstash, Kibana).

Outils utilisés

  • Cowrie : Honeypot SSH/Telnet haute interaction.
  • Docker : Conteneurisation.
  • Elastic Stack : Visualisation des attaques (Géolocalisation des IP).

Résultats

En 24 heures, plus de 500 tentatives de connexion ont été enregistrées. J'ai pu dresser une liste des identifiants (user/pass) les plus testés par les bots.
